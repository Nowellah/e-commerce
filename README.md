E-Commerce Website (Node.js)
A simple and modular e-commerce backend built with Node.js and Express.
This project includes essential features such as authentication, product management, cart operations, and order handling.

Features
User Authentication (Register / Login / JWT)
Product CRUD (Create, Read, Update, Delete)
Shopping Cart (Add / Remove Items)
Order Creation & Management
Basic Middleware for validation & security
Environment variable configuration using .env
RESTful API structure

Tech Stack
1. Node.js
2. Express.js
3. MongoDB 
4. JWT Authentication
5. Nodemon for development
